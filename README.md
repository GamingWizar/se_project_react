# Project 4: What to Wear

This is the forth project of the Web Development program at Triple Ten. It was created using HTML, CSS, JavaScript, and React, based on the design provided through Figma.

## Project features

- Semantic HTML5
- Flexbox
- Grid
- Flat BEM
- JS Classes
- Webpack bundling
- React
- API intergration
